# yeeway.dev
Alex Su's personal website

## Getting Started
```bash
git clone https://github.com/yeeway0609/yeeway.dev.git
cd yeeway.dev
bun i
bun dev
```
## Tech stack
- JS 框架：Next.js
- 樣式：Tailwind CSS、Shadcn-ui
- 內容：MDX
- 工具：ESlint
